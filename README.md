# 🎲 Yahtzee Scoreblad

Een digitaal Yahtzee scoreblad gebouwd met React en Vite. Ondersteunt 1 t/m 4 spelers, telt automatisch op en heeft een knop voor een nieuw spel.

## Snel starten

```bash
npm install
npm run dev
```

## Functies

- 1 tot 4 spelers met aanpasbare namen
- Automatisch optellen in real-time
- Bonusmeter voor bovenvak (+35 bij ≥ 63)
- Vaste scores met één klik (Full House, Straten, Yahtzee)
- Yahtzee Bonus teller (+100 per extra)
- Winnaar banner als iedereen klaar is
- Nieuw Spel knop
